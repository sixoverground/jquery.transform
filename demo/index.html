jQuery Transform

Hardware accelerated animation on iOS and Android — v0.1a

How to use

Include this script after jQuery.

Translate

Move an element to a new position.

Translate X

$('.box').transform({ x: '90px' });

Translate Y

$('.box').transform({ y: '30px' });

Translate X and Y

$('.box').transform({ translate: ['200px', '20px'] });

Rotate

Change the rotation of an element.

Rotate

$('.box').transform({ rotate: '30deg' });

Scale

Resize an element's width and height.

Scale Evenly

$('.box').transform({ scale: 1.2 });

Scale Horizontally

$('.box').transform({ scaleX: 2 });

Scale Vertically

$('.box').transform({ scaleY: 1.5 });

Scale Horizontally and Vertically

$('.box').transform({ scale: [2, 1.5] });

Skew

Skew an element on either axis.

Skew X

$('.box').transform({ skewX: '30deg' });

Skew Y

$('.box').transform({ skewY: '30deg' });

Skew X and Y

$('.box').transform({ skew: ['30deg', '-30deg'] });

Opacity

You can also animate regularly available CSS properties.

Opacity

$('.box').transform({ opacity: 0 });
