jQuery Transform

Hardware accelerated animation on iOS and Android — v0.1a

How to use

Include this script after jQuery.

Translate

Translate X

$('.box').transform({ x: '90px', duration: '1s' });

Translate Y

$('.box').transform({ y: '30px', duration: '1s' });

Translate X and Y

$('.box').transform({ 
  translate: ['200px', '20px'], 
  duration: '1s' 
});
